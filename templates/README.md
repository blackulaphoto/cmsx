# Templates Folder Guide

This folder is organized into three separate content types so the documentation module can distinguish real templates from reference material and AI instruction files.

## Document templates

Use these as selectable writing templates in the documentation workflow:

- `document-templates/Progress Report Template.md`
- `document-templates/Proof of Residence Template.md`
- `document-templates/Letter of Presence Template.md`
- `document-templates/Completion Letter Template.md`

These files are neutral placeholders. They do not contain real client data, hardcoded signatures, or embedded image blobs.

## Reference library

Use these as background guidance or compliance/reference context:

- `reference-library/Case Management Playbook.txt`
- `reference-library/ASAM Criteria.txt`

These are not end-user templates and should not appear in the template gallery as direct selectable documents.

## AI instructions

Use these only as internal AI behavior guidance:

- `ai-instructions/Brandon Workflow Lock.txt`

This file is prompt-style instruction content, not a client-facing or staff-facing document template.
