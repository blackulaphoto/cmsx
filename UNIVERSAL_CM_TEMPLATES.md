# UNIVERSAL CASE MANAGEMENT TEMPLATE LIBRARY
## Substance Use Disorder & Mental Health Treatment Documentation
### Version 1.0 | Facility-Agnostic | EHR-Ready Copy/Paste Format

---

## HOW TO USE THIS LIBRARY

**Placeholder convention** — Anything in `[BRACKETS]` is a fill-in. Replace before submitting.

**Universal placeholders used throughout:**
- `[FACILITY NAME]` — Your treatment center's legal/operating name
- `[FACILITY LEGAL ENTITY]` — Parent LLC / corporation if different
- `[FACILITY ADDRESS]` — Street, city, state, ZIP
- `[FACILITY PHONE]` — Main facility phone
- `[CM NAME]` — Case Manager full name
- `[CM CREDENTIALS]` — e.g., RADT, CADC, LCSW, etc.
- `[CM LICENSE #]` — State license / certification number
- `[CM EMAIL]` — Case Manager work email
- `[TH NAME]` — Primary Therapist name
- `[TH CREDENTIALS]` — e.g., AMFT, LMFT, LCSW, PsyD
- `[TH LICENSE #]` — Therapist license number
- `[PD NAME]` — Program Director
- `[CD NAME]` — Clinical Director
- `[MD NAME]` — Primary Physician
- `[CT NAME]` — Client / patient full name
- `[CT DOB]` — Date of birth
- `[CT MR#]` — Medical record number
- `[ADMIT DATE]` — Date of admission
- `[DC DATE]` — Date of discharge
- `[TODAY]` — Today's date

**Documentation rules that apply to every template below:**
1. Never fabricate quotes — preserve client statements verbatim
2. Never fabricate credentials, license numbers, or appointment details
3. Strengths-based, goal-oriented language (not deficit-focused)
4. No client names in clinical narrative ("Response") sections — use "client" or "CT"
5. Three-day signature protocol on items awaiting client signature; mark NS only after documented outreach

## 8.1 GROUP NOTE — VIRTUAL ATTENDANCE (BILLABLE)

Each group session generates one note per attending client. Mix engagement levels (~40% engaged with topic-related quote, ~60% non-participatory or partially engaged observations). Every note must be unique — no copy-paste duplicates.

**ENGAGED CLIENT — Template A**
```
Location of Client: [Sober Living / Home / Treatment Facility]. The client attended the group virtually via [Google Meet / Zoom]. The client displayed active listening and self-awareness, AEB maintaining eye contact, nodding in agreement, and responding respectfully to others. The client participated in each group activity and offered insight into their emotional progress. The client stated, "[VERBATIM TOPIC-RELATED QUOTE]"
```

**ENGAGED CLIENT — Template B**
```
Location of Client: [Sober Living / Home]. The client attended the group virtually via Google Meet. The client appeared content and at ease, smiling and contributing to the conversation with clear and concise responses. The client was able to share on the topic stating, "[VERBATIM TOPIC-RELATED QUOTE]"
```

**ENGAGED CLIENT — Template C**
```
Location of Client: [Sober Living / Home]. The client attended the group virtually via Google Meet. The client appeared calm and centered, AEB regulated tone, steady eye contact, and appropriate affect. The client shared voluntarily and stated, "[VERBATIM TOPIC-RELATED QUOTE]"
```

**NON-PARTICIPATORY — Template A**
```
Location of Client: [Sober Living / Home]. The client attended the group virtually via Google Meet. The client entered the group appearing disengaged, AEB avoiding eye contact, slouched posture, and minimal verbal participation. The client required multiple prompts to respond and provided brief, vague answers.
```

**NON-PARTICIPATORY — Template B**
